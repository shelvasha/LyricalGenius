LyricalGenius

get lyrics for your iTunes tracks

View on Github

Pre-requisites

The following are required for the applet to work:

  • subprocess
  • requests
  • bs4
  • BeautifulSoup

These can be install using pip install <package>. For more information, refer to pip documentation.

Usage

To use the applet, select a track in iTunes, and go to the iTunes scripts menu and select LyricalGenius.
